SDK - Software development kit
SDKs provide a comprehensive collection of tools that enable software developers to build software applications faster and in a more standardized way
Solution - קובץ קונפיגורציה שמאגד פרוייקטים לתוך קונטקסט. קונטיינר לפרוייקטים.

dotnet --info - מראה את כל גרסאות ההתקנה של ה.NET שנמצא על המחשב
dotnet new globaljson - קובץ שיתאים את הפרוייקט לגרסא הרצויה - לפתוח בתוך התיקייה שהפרוייקט ירוץ - .net sdk
dotnet new sln - יצירת soulotion בתיקייה שבה הפרויקט ירוץ
dotnet new webapi -o API
-o means output folder
dotnet add sln API - מוסיף את הפרוייקט של הAPI לתוך הSOULUTION

ctrl + shift + p - הוספת generate assest - מייצר קבצי הגדרות לדיבאג וראן 
dotnet dev-certs https --trust יצירת cerificate על המחשב שלנו בצורה ידנית .
